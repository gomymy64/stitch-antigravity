01
Создание приложения в Stitch

Открой Stitch и вставь промпт. Вместо apple.com подставь URL своего сайта - Stitch возьмёт оттуда визуал.

Промпт 1 - Stitch
Привет, хочу чтобы ты создал трёхстраничное мобильное приложение - Mastodon Hub. Это мой личный хаб. Первая страница - главная с новостями по нейросетям. Вторая - мои видео. Третья - форум, где можно пообщаться и задать вопросы. https://apple.com - возьми за основу визуал моего сайта и сделай всё на русском.
Вместо apple.com вставь URL своего сайта
02
Запуск в AntiGravity

Скачай zip-архив из Stitch, загрузи его в AntiGravity и попроси запустить.

Промпт 2 - AntiGravity
Привет, я загрузил код трёхстраничного мобильного приложения. Запусти его на localhost пожалуйста.
03
Установка скиллов

Подключи скиллы из GitHub-репозитория, чтобы Stitch мог работать через MCP.

Промпт 3 - Скиллы
Установи эти скиллы в репозиторий для Stitch: https://github.com/google-labs-code/stitch-skills
04
Stitch Loop - два проекта

Используй Stitch Loop для автоматического создания двух проектов одним промптом.

Промпт 4 - Stitch Loop
Используй Stitch MCP и Stitch Loop скилл. Создай два отдельных проекта в Stitch. Первый - для продажи африканского кофе. Второй - приложение Mastodon Hub, мой личный бренд-хаб: новости по нейросетям, мои видео, форум. Используй стиль с сайта apple.com. Для каждого - три страницы: главная мобильная, выбор контента, финальный экран. Всё на русском.
Вместо apple.com вставь URL своего сайта
05
Сборка и деплой

Когда дизайн готов - попроси AntiGravity собрать полноценное приложение и запушить на GitHub.

Промпт 5 - Деплой
Мне нравится дизайн из Stitch. Собери это в полноценное приложение и запушь на GitHub. Мобильное приложение, по тем брендбук-гайдлайнам что созданы в Stitch.
Настройка
Как подключить Stitch к AntiGravity через MCP
1
Заходишь на stitch.withgoogle.com → Settings → Create API Key → копируешь ключ
2
Открываешь AntiGravity → раздел MCP → находишь Stitch → вставляешь ключ
3
Готово - Stitch появится в списке MCP серверов с шестью скиллами
